Step 2 of 4

How it works

Simple process. Blind competition. Homeowner stays in control.

The flow

  1. Homeowner submits a request with property details, urgency, and project type.
  2. We match up to 3 contractors based on service area and fit.
  3. You submit a blind competitive bid without seeing anyone else’s price.
  4. The homeowner picks the contractor they want to move forward with.
  5. You install and complete the job directly with the homeowner.

The agreement

This is the fee structure and the operating rule set you’re agreeing to before account creation.

$800 per closed install job

  • ✓Fee amount: $800 per closed install job.
  • ✓When it’s owed: Only when the homeowner selects you and the install is completed.
  • ✓What counts as a closed job: A matched NeighborlyWork lead that becomes a completed HVAC install or replacement you perform for that homeowner.
  • ✓Billing timing: Billed after the install is completed, due within 30 days.
  • ✓If the job falls through: No fee if the homeowner never selects you, cancels before completion, or the install does not close.
  • ✓No extra charges: No fee for lost bids, no fee for unselected quotes, and no monthly charges.

You’ll be asked to explicitly agree to these terms on the next step before creating your account.
